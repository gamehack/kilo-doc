免费使用 Kilo

Kilo Code 可以完全免费使用。Kilo 在三个地方使用 AI 模型推理，每个都可以配置为使用免费模型。

Kilo 在哪里使用模型

  1. 智能体交互 — IDE 扩展（VS Code、JetBrains）、CLI 和云服务中的编程智能体对话
  2. 自动补全 — 输入时的编辑器内代码补全（仅 IDE 扩展）
  3. 后台任务 — 自动会话标题和上下文摘要

每个默认都会消耗额度。要完全免费使用 Kilo，将这三者都配置为使用免费模型。

免费智能体使用

Kilo 通过 Kilo Gateway 和合作伙伴提供商为编程任务提供免费模型。

Auto Free

最简单的方式是使用 Auto Free（kilo-auto/free）。这是一个 Kilo 提供的模型层级，自动将你的请求路由到最佳可用免费模型——无需配置。

⚠️ 数据处理

Auto Free 可能会将你的请求路由到记录提示和输出并用于改进其服务的提供商——包括 NVIDIA 的免费端点。使用 Auto Free 时请勿提交个人或机密数据。

查找其他免费模型

你还可以浏览并选择单独的免费模型。在模型选择器中输入 free 来过滤列表——免费模型在所有平台上都有清晰标记。

免费自动补全

通过 BYOK（自带密钥） 在 Kilo Gateway 上添加你自己的 Mistral AI（Codestral）API 密钥。Mistral 为 Codestral 提供免费层级。

免费后台任务

将小型模型设置为免费模型即可避免后台任务的额度消耗。在 VS Code 扩展中：进入 设置 → 模型，将小型模型更改为任何免费模型。
