Kilo Code 入门

Kilo Code 是一个开源的 AI 编程助手，可以在你工作的任何地方使用——IDE、终端、浏览器或移动端。通过理解你代码库的 AI 来生成代码、自动化审查、调试问题并更快地交付。

在哪里使用 Kilo

  • 在你的 IDE 中 — VS Code、JetBrains、Cursor、Windsurf 以及其他 VS Code 分支
  • CLI — 从终端运行 Kilo，用于脚本和自动化
  • Web 和移动端 — 从浏览器访问 Kilo（即将推出）或 iOS/Android 应用
  • Slack — 直接在你的工作空间中与 Kilo 聊天

你的会话在所有这些平台之间同步，因此你可以在手机上开始一个任务，然后在 IDE 中完成它。

Kilo 能做什么

  • AI 编码 — 通过自然对话生成、重构和调试代码。使用专门的智能体（Code、Architect、Debug、Ask）或创建你自己的。通过 Autocomplete 获取内联建议。
  • 协作 — 共享会话、管理团队设置、跟踪组织内的 AI 采用情况。
  • 自动化 — 设置 AI 驱动的代码审查、分类智能体和自动修复器。
  • 部署与安全 — 直接从 Kilo 构建和部署应用。运行安全扫描并通过 AI 辅助管理问题。

快速开始

  1. 安装 Kilo Code
  2. 连接 AI 提供商 或使用 Kilo 内置提供商和额度
  3. 运行你的第一个任务
💡 提示

配置 Kilo 最简单的方式是让智能体来帮你。只需告诉智能体你想要什么——"添加这个 MCP 服务器"、"禁用 OpenAI"、"添加我的 Ollama 端点"。智能体有一个内置技能，可以读取和更新你的 kilo.jsonc 配置。

刚接触 AI 编程助手？在了解 Kilo 本身的功能之前，你可以先在 path.kilo.ai 了解智能体工程。
