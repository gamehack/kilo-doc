安装

在你首选的平台上安装 Kilo Code 开始使用。在下方选择你的开发环境：

当前的 Kilo Code 扩展基于 Kilo CLI 构建，以预发布版本的形式在 VS Code Marketplace 上分发。

  1. 打开 VS Code
  2. 进入扩展面板（Ctrl+Shift+X / Cmd+Shift+X）
  3. 搜索 "Kilo Code"
  4. 点击 安装 旁边的下拉箭头，选择 安装预发布版本
ℹ️ 信息

"预发布"标签是 VS Code Marketplace 的分发渠道——扩展是稳定的，推荐所有用户使用。

通过 npm 安装：

npm install -g @kilocode/cli

验证安装：

kilo --version

首次设置：安装后运行 kilo 并使用 /connect 命令添加你的第一个提供商凭据。

Kilo Code 支持所有 JetBrains IDE，包括 IntelliJ IDEA、WebStorm、PyCharm 等。

  1. 打开你的 JetBrains IDE
  2. 进入 Settings/Preferences → Plugins
  3. 点击 Marketplace 标签
  4. 搜索 "Kilo Code"
  5. 点击 安装 并重启 IDE

支持的 IDE：IntelliJ IDEA、WebStorm、PyCharm、PhpStorm、GoLand、Rider、CLion、RubyMine、DataGrip。Community 和 Ultimate 版本均支持。

通过 Slack 与 Kilo 交互——直接在你的工作空间中与 AI 编程助手聊天。

详见 Slack 平台 页面。

手动安装

通过 Open VSX Registry

Open VSX Registry 是一个开源的 VS Code Marketplace 替代方案。

  1. 打开你的编辑器
  2. 访问扩展视图
  3. 搜索 "Kilo Code"
  4. 选择 "Kilo Code" 并点击 安装
  5. 如果提示，重新加载编辑器

通过 VSIX 文件

  1. 从 GitHub Releases 下载 .vsix 文件
  2. 在 VS Code 中，打开扩展视图
  3. 点击 "..." 菜单
  4. 选择 "Install from VSIX..."
  5. 浏览并选择下载的 .vsix 文件

故障排除

扩展不可见

  • 重启 VS Code
  • 验证 Kilo Code 在扩展中已列出并启用
  • 尝试禁用并重新启用扩展
  • 检查输出面板是否有错误（View → Output，选择 "Kilo Code"）

安装问题

  • 确保网络连接稳定
  • 验证 VS Code 版本 1.84.0 或更高
  • 如果 VS Code Marketplace 不可访问，尝试 Open VSX Registry 方法

Windows 用户

确保 PowerShell 已添加到 PATH：

  1. 打开 编辑系统环境变量 → 环境变量
  2. 在 系统变量 下，选择 Path → 编辑 → 新建
  3. 添加：C:\Windows\System32\WindowsPowerShell\v1.0\
  4. 点击 确定 并重启 VS Code
