快速开始

在 设置 Kilo Code 之后，按照以下步骤操作。

步骤 1：打开 Kilo Code

点击 VS Code 主侧边栏中的 Kilo Code 图标打开聊天面板。如果没有看到图标，验证 扩展已安装。

步骤 2：输入你的任务

在聊天框中输入你想要 Kilo Code 做什么的清晰描述。以下是一些示例：

  • "创建一个名为 hello.txt 的文件，内容为 'Hello, world!'"
  • "编写一个 Python 函数，实现两个数相加"
  • "创建一个简单的 HTML 网站，标题为 'Kilo 测试'"

不需要特殊命令或语法——只需使用自然语言。

步骤 3：发送任务

按 Enter 发送。

步骤 4：审查并批准操作

Kilo Code 分析你的请求并提出操作建议。默认情况下，大多数工具是自动批准的——只有 shell 命令、外部目录访问和敏感文件读取会提示确认。

要更改哪些操作需要批准，打开 设置（齿轮图标）并进入 自动批准 标签。你可以将每个工具设置为允许、询问或拒绝。

步骤 5：迭代

Kilo Code 以迭代方式工作。继续提供反馈或后续指令，直到你的任务完成。

💡 提示

加速开发：检出多个代码仓库副本并并行运行 Kilo Code（使用 git 解决冲突，与人类开发者相同）。这可以显著加速大型项目的开发。

结论

你已完成第一个任务。在此过程中你学到了：

  • 如何使用自然语言与 Kilo Code 交互
  • 为什么批准机制让你保持控制
  • 迭代如何让 AI 完善其工作

准备好了吗？以下是一些后续步骤：

  • 智能体 — 探索不同任务的不同智能体
