与 Kilo Code 聊天

核心要点：Kilo Code 是一个 AI 编程助手。你用自然语言与它聊天，它为你编写、编辑和解释代码。

快速设置

点击 VS Code 主侧边栏中的 Kilo Code 图标打开侧边栏聊天。你也可以将其弹出到编辑器标签页中以获得更大的工作空间。

如何与 Kilo Code 对话

关键洞察：只需用普通语言输入你想要的内容。不需要特殊命令。

好的请求示例：

  • "创建一个名为 utils.py 的新文件，添加一个名为 add 的函数，接收两个数字参数并返回它们的和"
  • "在文件 @src/components/Button.tsx 中，将按钮颜色改为蓝色"
  • "找到 @/src/App.js 中变量 oldValue 的所有实例，替换为 newValue"

让请求有效的关键：

  • 具体明确 — "修复 calculateTotal 中返回不正确结果的 Bug" 优于 "修复代码"
  • 使用 @ 引用 — 用 @filename 直接引用文件和代码
  • 一次一个任务 — 将复杂工作分解为可管理的步骤
  • 包含示例 — 展示你想要的风格或格式

聊天界面

  • 输入提示 — 输入请求并按 Enter 发送
  • 操作按钮 — 批准或拒绝建议的更改
  • 智能体下拉菜单 — 从侧边栏切换智能体
  • 会话管理 — 开始新会话或恢复之前的会话
