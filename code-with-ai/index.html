AI 编码

Kilo Code 是你的 AI 结对程序员，在你的 IDE、终端或浏览器中工作。通过理解你代码库和上下文的 AI 来生成代码、重构、调试并更快地交付。

快速开始

刚接触 Kilo Code？从这里开始了解核心概念：

  • 安装 Kilo Code — 在 VS Code、JetBrains、CLI 或移动端开始
  • 连接 AI 提供商 — 设置你喜欢的模型
  • 快速开始指南 — 几分钟内运行你的第一个任务

平台

在你工作的任何地方使用 Kilo Code：

  • VS Code — 最流行的 IDE 集成
  • JetBrains — IntelliJ、PyCharm、WebStorm 等
  • CLI — 基于终端的 AI 编程
  • Cloud Agent — 在云端运行 Kilo
  • 移动应用 — iOS 和 Android 支持
  • Slack — 在你的工作空间中与 Kilo 聊天

智能体

Kilo 使用专门的智能体来帮助完成不同的任务：

  • 聊天界面 — 基于对话的编程
  • 使用智能体 — 在 Code、Ask、Plan、Debug 等智能体之间切换
  • 模型选择 — 为每个任务选择合适的 AI 模型
  • 上下文引用 — 引用文件、函数和符号

功能

  • 自动补全 — 输入时的内联代码建议
  • 快速编辑 — 快速文件修改
  • 代码操作 — AI 驱动的重构和修复
  • 任务与待办列表 — 分解复杂任务
  • 检查点 — 保存和恢复工作状态
  • 浏览器使用 — 自动化 Web 交互
  • 增强提示 — 自动改进你的提示
  • Git 提交生成 — AI 驱动的提交消息
